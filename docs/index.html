AnimateForms Documentation

Overview

SDProductions does not recommend the 'plug and play' use of this library in production environments yet. We test our software to an extent of basic use, but as 'we' are a one person team so we don't test everything. Edge cases may not be covered. Please also test your software before deploying. You have been warned.

Classes Included

Animate.cs

An instantiable (but only one is needed) class to facilitate with animation.

Easings.cs

Static class with all easing functions from Gizma.

Helpers.cs

Static class with functions to assist with using the API.

Options.cs

Instantiable options class to assist with using the API.
